# Introdução

Nesta seção, são apresentadas a modelagem de dados e a arquitetura de comunicação que sustentam o funcionamento do sistema AMORA. Para garantir a organização, a consistência e a integridade das informações, este capítulo detalha tanto a estrutura do banco de dados relacional quanto a especificação dos endpoints da API. A documentação a seguir descreve como os dados de usuários, solicitações, tarefas e notificações são armazenados, manipulados e trafegados entre a interface visual (frontend) e o servidor da aplicação (backend).
