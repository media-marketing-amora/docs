# Restrições

## RES01 – O núcleo visual do sistema exige o runtime de JavaScript, Node.js na versão 18.0.0 ou superior (preferencialmente versão LTS). 

- RES01.01 – React: Biblioteca base para a construção da interface de usuário reativa, modular e baseada em componentes; 

- RES01.02 – TypeScript: Linguagem base para adição de tipagem estática ao código JavaScript, garantindo maior confiabilidade e reduzindo erros em tempo de compilação; 

- RES01.03 – Vite: Ferramenta de compilação e servidor de desenvolvimento de alta performance para o ecossistema frontend; 

- RES01.04 – Tailwind CSS: Framework focado em classes utilitárias para a estilização rápida, responsiva e padronizada de acordo com o sistema de design corporativo; 

- RES01.05 – DnD Kit: Conjunto de bibliotecas para a implementação da funcionalidade de drag and drop (arrastar e soltar) no quadro Kanban; 

- RES01.06 – Axios: Cliente HTTP para consumo da API e integração de dados com o servidor de backend; 

- RES01.07 – Lucide React: Biblioteca de ícones vetoriais leves para a composição dos elementos visuais da interface; e 

- RES01.08 – Oxlint: Ferramenta de análise estática de código (linter) de alto desempenho para garantia das boas práticas e padronização do desenvolvimento. 
 

## RES02 – O núcleo lógico do sistema exige a linguagem de programação Python na versão 3.14 ou maior, incluindo as dependências: 

- RES02.01 – FastAPI: para a criação e gerenciamento das rotas (endpoints); 

- RES02.02 – Uvicorn: para a execução do servidor web; 

- RES02.03 – Pydantic-settings: para a obtenção das variáveis de ambiente do projeto diretamente em código Python; 

- RES02.04 – SQLAlchemy: para a criação dos modelos e manipulação dos registros do banco de dados diretamente em Python; 

- RES02.05 – Alembic: para o controle e estruturação das migrações do banco de dados de acordo com os modelos definidos no SQLAlchemy; 

- RES02.06 - Aiosqlite: para garantir que a conexão com o banco de dados seja realizada de forma 100% assíncrona; e 

- RES02.07 – UV: para a o gerenciamento das dependências e configurações do projeto.
